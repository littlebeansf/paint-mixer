# PaintMixer — Professional Color Mixing Studio

An interactive web application for painters and decorators to mix colors and preview professional special effects.

## Features

### Color Mixer
- Mix up to **8 pigments** simultaneously with adjustable ratios
- Physically-accurate color mixing using linear RGB blending
- Live HEX, RGB, and HSL values with one-click copy
- Random mix generator
- Save mixes to your palette

### Design
- Dark / Light mode toggle
- Animated effect previews (Kristall, Metallic, Pearl, Glitter, Mica, Galaxy…)
- Smooth transitions and micro-interactions
- Fully responsive (mobile, tablet, desktop)
